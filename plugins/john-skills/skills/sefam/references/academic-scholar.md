# Style: Academic Scholar

**Feel:** Scholarly serif on parchment + charcoal. Calm, credible, careful. The safe default for a manager audience.

**Best for:** serious architecture explainers, design decisions, anything that should read as considered and trustworthy.

**Build prompt:**
- Background: warm off-white / parchment (e.g. `#f5f1e8`). Body text charcoal (`#2b2b2b`), not pure black.
- One deep accent only — oxblood (`#6b2a2a`) or navy (`#1f3a5f`) — for section heads, rules, and the `<h1>`.
- Serif body: `Georgia, "Times New Roman", serif`. Real body size ~1.1rem, line length ~60–70ch, generous margins.
- Section heads: small-caps or a drop-cap on the opening paragraph. Separate sections with hairline rules (`border-top: 1px solid` in a muted tone).
- No shadows, no gradients, no rounded cards. Let typography and whitespace carry it.
- Light-only design — do not force theme-switching.
