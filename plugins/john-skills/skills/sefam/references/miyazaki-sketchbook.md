# Style: Miyazaki Sketchbook

**Feel:** Hayao Miyazaki storyboard — soft watercolor washes, airy, warm, calm, human.

**Best for:** a gentle, human explainer; a concept you want to feel approachable and unhurried.

**Build prompt:**
- Soft watercolor-wash background using gentle CSS gradients — sky-blues (`#bcd8e8`) into greens (`#cfe3c4`) and cream.
- Hand-drawn feel: gentle rounded shapes, soft edges, no hard borders.
- Airy whitespace — let the page breathe. Nothing crowded.
- Warm, calm type: a humanist sans or soft serif — `"Segoe UI", system-ui, sans-serif` or `Georgia, serif`.
- Muted, natural accent colors for section heads (a leaf green or sky blue). No loud saturation.
- Optional faint paper texture via a very subtle CSS pattern.
- Body text ~1.1rem, comfortable line length. Warmth must not cost legibility.
- Light-only design.
