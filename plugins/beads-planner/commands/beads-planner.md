# Beads Planner

Convert a GitHub issue into a structured Beads work breakdown.

Invoke the `issue-to-beads` skill with the user's argument:

```
Skill: beads-planner:issue-to-beads
```

Pass the GitHub issue reference provided by the user: $ARGUMENTS
